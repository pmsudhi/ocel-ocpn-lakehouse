# Production System Summary - OCEL/OCPN Lakehouse

## Executive Overview

The OCEL/OCPN Lakehouse is a production-ready data lakehouse system built on Apache Iceberg with Daft for process mining analytics. This system provides comprehensive support for Object-Centric Event Logs (OCEL) and Object-Centric Petri Nets (OCPN) with advanced analytics capabilities.
